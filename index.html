الاسم ثلاثي

أتعهد بعدم الأستعانه بأي مصدر خارجي وأن أتم حل الاسئله بمجهودي .

برجاء ادخال الاسم

السؤال الأول

يقول الله تعالى { وَالْعَادِيَاتِ ضَبْحًا } ما معني ضَبْحًا ؟

السؤال الثاني

يقول الله سبحانه في وصف جزاء المتقين" { إِنَّ لِلمُتَّقينَ مَفازًا حَدائِقَ وَأَعنابًا وَكَواعِبَ أَترابًا وَكَأسًا دِهاقًا } ما معني كأسا دهاقا ؟

السؤال الثالث

من اوصاف القرآن انه ؟

السؤال الرابع

في قوله تعالي { بَل كَذَّبوا بِالحَقِّ لَمّا جاءَهُم فَهُم في أَمرٍ مَريجٍ } ما معني أمر مريج ؟

السؤال الخامس

يقول الله تعالى في وصفه نعيم أهل الجنة { مُتَّكِئينَ عَلى رَفرَفٍ خُضرٍ وَعَبقَرِيٍّ حِسانٍ } ما معني عبقري حسان ؟

السؤال السادس

ما معني قوله تعالي { لِإِيلَافِ قُرَيْشٍ } .

السؤال السابع

قال تعالى { تُسْقَىٰ مِنْ عَيْنٍ آنِيَةٍ } ما معني عَيْنٍ آنِيَةٍ ؟

السؤال الثامن

قالى تعالى { فَلَمَّا دَخَلُوا عَلَيْهِ قَالُوا يَا أَيُّهَا الْعَزِيزُ مَسَّنَا وَأَهْلَنَا الضُّرُّ وَجِئْنَا بِبِضَاعَةٍ مُّزْجَاةٍ } ما معني مُّزْجَاةٍ ؟

السؤال التاسع

قال تعالى { وَالذَّارِيَاتِ ذَرْوًا (*) فَالْحَامِلَاتِ وِقْرًا } والْحَامِلَاتِ هي السحب فما معني وِقْرًا ؟

السؤال العاشر

القرآن مليان أدعية جميلة وجامعة لكل الخير إكتب أي دعاء منهم  يجي في بالك .

سبيلي اي حاجه عايزه تقوليها
او اقتراح لمسابقه السنه الجايه ان شاء الله
